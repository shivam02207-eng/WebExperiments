# 🌐 Web Development Experiments Repository

## 📌 Overview

This repository contains a collection of **Web Development experiments and projects** created as part of academic coursework.
It covers fundamental to intermediate concepts of web development including frontend and backend technologies.

---

## 🎯 Aim

To understand and implement core concepts of web development such as:

* HTML structure
* CSS styling
* JavaScript functionality
* Flask (Python backend)
* CRUD operations

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **Python (Flask)**
* **SQLite (for database projects)**

## 🚀 How to Run Projects

### ▶️ For HTML/CSS/JS Projects

1. Download or clone the repository
2. Open the `.html` file in your browser

---

### ▶️ For Flask Projects

1. Install Flask:

```
pip install flask
```

2. Run the application:

```
python app.py
```

3. Open browser and go to:

```
http://127.0.0.1:5000/
```
